class KnobEvent: (source)

View In Hierarchy

This dataclass represents the event generated by the Knob control.

Class Variable direction Knob direction event. 1 for up, -1 for down, 0 for no change.
Class Variable inverse_unit_value Knob inverse unit value event. Knob position reversed, but in unit form from 0 - 1.0.
Class Variable inverse_value Knob inverse value event. Knob position reverse, so value of 127 is 0 and 0 is 127.
Class Variable symmetry_unit_value Knob symmetry unit value event. Knob position represented in symetry from -1.0 - 1.0
Class Variable symmetry_value Knob symmetry value event. Knob position represented in symetry from -64 - 64.
Class Variable unit_value Knob unit value event. Knob position represented as a float from 0 - 1.0.
Class Variable value Knob value event.
