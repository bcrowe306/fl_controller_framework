class KnobControl(Control): (source)

Constructor: KnobControl(name, channel, identifier, status, ...)

View In Hierarchy

This control type is for knobs on your midi controller. It accepts the same parameters as the base Control Type, but emmits special events along with the value of the fader. Events: * unit_value: This event is emitted with the fader position represented as a float from 0 - 1.0. * inverse_value: this event is emitted with fader position reverse, so value of 127 is 0 and 0 is 127. * inverse_unit_value: this event is emitted with fader position reversed, but in unit form from 0 - 1.0. * symentry_value: this event is emitted with the fader position represented in symetry from -64 - 64. * symentry_unit_value: this event is emitted with the fader position represented in symetry from -1.0 - 1.0
