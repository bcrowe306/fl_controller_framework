fl_controller_framework.modes.Mode

class documentation

class Mode(EventObject, StateBase): (source)

Constructor: Mode(name, components, active_color, inactive_color, ...)

View In Hierarchy

This class represents a Mode and is used to give your controller varying functionality with the same controls. Modes can have multiple components attached to them and controls that active the mode.

Method __init__ Undocumented
Method activate Activate all components. This method goes though the self.components list and runs that activate method on each component.
Method deactivate Deactivate all components. This method goes though the self.components list and runs that deactivate method on each component.
Instance Variable active_color This is the name of the color sent to the control when the mode is active. This is used to send lighting to your controller to show which mode is active.
Instance Variable components A list of components to activate or deactivate with the mode.
Instance Variable inactive_color This is the name of the color sent to the control when the mode is inactive. This is used to send lighting to your controller to show which mode is inactive.
Instance Variable name Name must be unique to for Modes component it will exist.
