class GlobalEventObject(EventObject): (source)

Constructor: GlobalEventObject(*args, **kwargs)

View In Hierarchy

This is a global event registry. This class inherits from EventObject and is used to receive the events from FL Studio. It is a singleton object, and is attached to Component, and Control classes as instance variables. This allows Component, Controls, and Control Surface to subscribe-to and react to FL Studio events.
