class DynamicClassAttribute: (source)

Constructor: DynamicClassAttribute(fget, fset, fdel, doc)

View In Hierarchy

Route attribute access on a class to __getattr__. This is a descriptor, used to define attributes that act differently when accessed through an instance and through a class. Instance access remains normal, but access to an attribute through a class will be routed to the class's __getattr__ method; this is done by raising AttributeError. This allows one to have properties active on an instance, and have virtual attributes on the class with the same name (see Enum for an example).
